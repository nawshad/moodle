Installation:

1) copy ./lambdasso to ./auth/

Moodle 2.x will install the plugin automitically. If it does not visit your site and the admin page.

eg. http://example.com/moodle/admin/

2) Enable plugin 

Site Administration -> Plugins -> Authentication -> Manage Authentication

3) Configure the plugin.

In the same section as above, click on Settings for Lambda SSO.

Enter the fields and save.
